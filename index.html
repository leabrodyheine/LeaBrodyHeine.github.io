Superhero Escape

The goal: help the superhero safely make it back to her computer at the end of the maze.

To navigate this game, simply use the following keys:

  • A: will move the player one square to the left
  • S: will move the player one square down
  • D: will move the player one square to the right
  • W: will move the player one square up

Landing on a ticket will fully replenish the player's stamina (displayed by the green bar on the right). If the player lands on a popcorn, it will add +10 back to the stamina.

Food Desert Analysis

For this project, I looked at the density of grocery stores in different states and counties in the USA. The goal of my summary-generator function was to figure out how to use the function and the input tables to produce an output table with only these columns: “state”, “abbv”, “population”, and “store-summary”. The "store-summary" column summarizes some statistics about the total number of stores (grocery and convenience) across counties in that state, based on the summary function input. The statistic might be the total, average, median, etc. Throughout the process, numerous other tables and graphs are produced.

I was then able to answer several questions regarding my findings on food deserts:

  1. states have the highest variability (measured using standard deviation) of stores across counties?
  2. Do states with the largest populations also have the most combined stores?
  3. Are counties with the largest populations in the states with the largest number of stores per capita?

Some key skills learned:

  • Data Cleaning
  • Complex, multi-step & multi-function process
  • Understanding how libraries (such as PANDAS) are built
  • Using real-world data to ask and answer questions that have an impact on communities

Text Alignment

The purpose of this project was to showcase my skills in object-oriented programming and Java. The project reads a .txt file and then depending on the command line input, aligns it correctly given the specified maximum line length.

Alignment types are:

  • left
  • right
  • center
  • justify
  • triangle

The command line arguments are java TextAlignment '.txt file name' 'alignment type' then 'maximum line length'.
